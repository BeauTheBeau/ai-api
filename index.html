AI API Documentation

Routes (v1)

Locally Processed
Moderate Speed
Decent Results

Utilizing the LanguageModels library for locally processed, moderately-paced results.

  • /generate/{prompt} Generate text from a prompt
  • /generate/{prompt}/{context} Generate text from a prompt and context

Routes (v2)

Locally Processed
Fast
Fair Results

Employing DialoGPT for quick, though slightly less accurate, locally processed responses.

  • /generate/v2/?prompt={prompt} Generate text from a prompt

Routes (v3)

High-Quality Results
Fast
Remotely Processed

Utilizing OneAI, including GPT-3.5 Turbo, for rapid, high-quality responses through remote processing.

Note that OneAI has a 100,000-word limit for the Free Tier.

  • /generate/v3/?prompt={prompt} Generate text from a prompt
  • /generate/v3/?prompt={prompt}&context={context} Generate text from a prompt and context
